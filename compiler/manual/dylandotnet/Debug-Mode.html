2.4 Debug Mode

The #debug statement is used to turn on/off the production of debugging symbols in ‘.mdb’ (Mono) or ‘.pdb’ (.NET) format. By default debug mode is off. Turning the debug symbol making on will also imply #define DEBUG. For more info on the latter, see Conditional Compilation. To turn debug symbol making on, use #debug on. This statement should occur before the assembly and ver statements.
