An array of string with the arguments. Currently only 1 element is required. i.e. the path to the main file specifying other files etc. Further elements in the array will be left uninterpreted.

⊟ Remarks

If invoking programmatically please do not use Process.Start() but reference dnc.exe and invoke dnc.main(string[]) directly. If using the console type 'dnc [path]'.
