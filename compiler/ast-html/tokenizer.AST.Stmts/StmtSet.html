StmtSet Class

Represents a set of statements in the dylan.NET language.

public class StmtSet

⊟ Remarks

Nothing enforces an 'StmtSet' i.e. a statement set to contain a whole file's worth of statements. This type is for moving statments around in one convenient package. This class is mainly for infrastructure use by the compiler components.
