IAsyncValue<TResult> Generic Interface

An interface for the mechanism with which a value can be asynchronously returned from an awaited method.

Syntax

public interface IAsyncValue<TResult>

Type Parameters

TResult
The type of the value returned.

Remarks

This library provides a reference implementation of this interface. The methods of this library depend on this interface, not on the implementation provided by this library to enable consumers to implement the interface in the way that is best for their application.

IsAvailable Property

Gets whether the value is available.

Syntax

public bool IsAvailable { get; }

Value

True if the value is available, else false.

Remarks

In the case where an Await overload that returns a value inside an IAsyncValue is used, you can use this property to see if the await actually happened successfully if the await was part of an optional execution path.
