DisposerTaskClosure<T,TDisposable> Generic Class

A base class for closures used to implement asynchronous methods that also disposes a resource when the asynchronous activity is done/fails.

Syntax

public class DisposerTaskClosure<T, TDisposable> : TaskClosure<T>
where TDisposable : class, IDisposable

Type Parameters

T
The return type for the asynchronous activity.
TDisposable
The type of disposable resource that should be managed by the closure.

Remarks

This class is based on TaskClosure<T> and overrides its Final method to dispose the passed in resource.
