IAwaiter Interface

An interface that formalises what it means to be awaiter.

Syntax

public interface IAwaiter : System.Runtime.CompilerServices.INotifyCompletion

Remarks

Implement this interface to allow your objects to be awaited by this library. You must also implement INotifyCompletion.

GetResult Method

Concludes the await, throwing any exceptions thrown by the awaitable in the process.

Syntax

public void GetResult ()

Remarks

This library runs this method in its completion handlers. It runs a continuation given by the user if no exceptions are raised, else runs the user provided exception handler with the raised exception.

IsCompleted Property

Gets the completion state of the awaitable being awaited.

Syntax

public bool IsCompleted { get; }

Value

True if the completion handler can be run immediately, else false.

Remarks

This library runs the completion handler immediately if this is true else subscribes the handler using OnCompleted.
