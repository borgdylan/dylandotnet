AsyncHelpers Class

A set of helpers for iterating through a list that yields values asynchronously using synchronous or asynchronous handlers.

Syntax

public static class AsyncHelpers

Remarks

This class provides an implementaion of the "await foreach" pattern for languages that lack a built-in equivalent. Collections to be iterated must implement IAsyncEnumerable<T> or IAsyncEnumerator<T>.

AwaitForeach<TElement> Generic Method

Iterates through an asynshcronous enumerable collection using the provided synchronous handler.

Syntax

public static Task AwaitForeach<TElement> (IAsyncEnumerable<TElement> alst, Func<TElement, bool> act)

Type Parameters

TElement
The type of the elements in the asynchronous enumerable.

Parameters

alst
The asynchronous enumerable to iterate through.
act
A handler that gets called for each element in the collection. The return value must be true to keep iterating through the rest of the list, false to break the loop.

Returns

A Task that completes when the loop terminates either as the sequence gets exhausted or because the handler signals that it wants an early termination.

Remarks

Do not forget to await the returned task. This makes sure that the loop would be terminated and that any enumerators would have been disposed before continuing.
