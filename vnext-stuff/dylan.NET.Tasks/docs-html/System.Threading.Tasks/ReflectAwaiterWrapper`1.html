ReflectAwaiterWrapper<TResult> Generic Class

An IAwaiter<TResult> implementation that wraps an ordinary object.

Syntax

public class ReflectAwaiterWrapper<TResult> : System.Runtime.CompilerServices.INotifyCompletion, IAwaiter<TResult>

Type Parameters

TResult
The type of result returned by the awaitable being awaited.

Remarks

This implementation makes use of reflection and is not available for PCLs. It is recommended to make use of strongly typed wrappers instead due to the penalties associated with reflection. If an IAwaiter<TResult> implementation is passed in, no reflection will be used in preference to calling the methods on the IAwaiter<TResult> directly.
