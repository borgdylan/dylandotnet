TaskHelpers Class

Helpers for awaiting Tasks and other things that implement IAwaiter or IAwaiter<T>.

Syntax

public static class TaskHelpers

Remarks

These helpers may also be useful for other languages that do not support the await keyword. Do note that this API surface is also available on .NET 4 and some PCL profiles.

Await the awaitable linked to the provided awaiter using the given functions.

Syntax

public static Task<U> Await<T, U> (IAwaiter<T> a, Func<T, U> f, Func<Exception, U> cat, System.Threading.CancellationToken token)

Type Parameters

T
The type of result returned by the awaitable being awaited.
U
The type of result returned by the awaiting functions.

Parameters

a
The awaiter to use.
f
The code to execute once the await finishes successfully.
cat
The code to execute if the await or the code that executes after it, terminates with exceptions.
token
The cancellation token to use.

Returns

A new task representing the await.

FromResult<TResult> Generic Method

Returns an completed task of the given type with the given result value.

Syntax

public static Task<TResult> FromResult<TResult> (TResult result)

Type Parameters

TResult
The result type desired for the returned Task.

Parameters

result
The value to package into a Task<TResult>.

Returns

A completed Task<TResult> with the specified result.

Remarks

Note that this method is only defined on .NET 4 and some of the supported PCL profiles. For .NET 4.5, ASP.NET vNext on Desktop/Core CLR and the other supported profiles use the FromResult<TResult> hanging off the Task class. This method is therefore a polyfill for profiles that do not provide it.
