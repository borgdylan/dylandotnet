TaskClosure Class

A base class for closures used to implement asynchronous methods.

Syntax

public class TaskClosure : TaskClosure<AsyncVoid>

Remarks

This class encapsulates a TaskCompletionSource instance and calls methods on it as you do awaits or return a value. It also handles exceptions thrown by awaited methods and sets them on the Task exposed by the TaskCompletionSource.
