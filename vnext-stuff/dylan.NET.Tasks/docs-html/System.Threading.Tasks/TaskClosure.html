TaskClosure Class

A base class for closures used to implement asynchronous methods.

Syntax

public class TaskClosure

Remarks

This class encapsulates a TaskCompletionSource instance and calls methods on it as you do awaits or return a value. It also handles exceptions thrown by awaited methods and sets them on the Task exposed by the TaskCompletionSource.

Canceller Property

The cancellation token used to cancel an asynchronous method at await points.

Syntax

public System.Threading.CancellationToken Canceller { get; set; }

Value

The cancellation token set by the user or CancellationToken.None if no token was set.

Remarks

The token can be changed in between awaits. This property is read before each await happens.
