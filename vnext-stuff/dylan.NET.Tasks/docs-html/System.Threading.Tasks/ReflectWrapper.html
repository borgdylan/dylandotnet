ReflectWrapper Class

An IAwaitable implementation that wraps an object.

Syntax

public class ReflectWrapper : IAwaitable

Remarks

This implementation makes use of reflection and is not available for PCLs. It is recommended to make use of strongly typed wrappers instead due to the penalties associated with reflection. If an IAwaitable implementation is passed in, no reflection will be used in preference to calling the methods on the IAwaitable directly.
