dylan.NET.Tasks

Task Helpers for dylan.NET

System.Threading.Tasks Namespace

Helpers for awaiting stuff and implementing asynchronous routines mostly targeted for dylan.NET projects on .NET 4, .NET 4.5, ASP.NET vNext and PCLs.
