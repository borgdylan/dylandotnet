dylan.NET.Tasks

Task Helpers and awaiting support for dylan.NET and other languages without support for native async/await.

System.Threading.Tasks Namespace

Helpers for awaiting tasks etc. and implementing asynchronous routines mostly targeted for dylan.NET projects on .NET 4+, .NET Core, and .NET Standard.
